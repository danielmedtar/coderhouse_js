Storage

El setItem recibe 2 parámetros. Ambos string

Para manejar la clave de los parámetros correctamente, la clave (key) debe colocarse entre comillas. El valor se coloca con el nombre de la variable

La información del Local Storage persiste cuando se cierra la ventana, y el session no se mantiene.

Para los carritos de compra por ejemplo, lo mejor es almacenar lo que coloca el usuario en local storage para que si actualiza no se pierda lo que ya ha hecho

Tampoco es recomendable usar el localStorage para todo e indefinidamente. En algún momento hay que limpiar el codigo
